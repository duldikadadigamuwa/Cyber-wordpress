u# Cyber — Tech Shop (meyoro.com)

This repository contains the WordPress code for **Cyber**, a modern tech e-commerce site for gadgets, laptops, phones, PC parts, and accessories.  
**Live site:** https://meyoro.com

> ⚠️ This repo includes theme/plugin code and WordPress core but **excludes** sensitive files (`wp-config.php`, `.htaccess`) and the database export.

---

## Highlights

- **Product catalog** across key categories (phones, laptops, accessories, processors, and more)
- **E-commerce UX**: add to cart, quick view, and wishlist
- **Wishlist** page to save items for later
- **Order tracking** by order ID and billing email
- **SEO-friendly structure** with clean category/brand pages and product detail URLs
